## 使用说明

这是一个示例 Skill，用于验证 LiteClaw 的 skill 加载是否正常。

- 若看到此内容被注入到 system prompt，说明 skill 机制工作正常
- 可参考此格式创建自己的 skills，或从 OpenClaw 社区获取 skills 放入 skills 目录

## 示例任务

- 用户说「给我一个示例」时，可简要说明 skill 机制
- 用户说「测试 skill」时，可确认 skill 已加载
